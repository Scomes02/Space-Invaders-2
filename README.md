# Space-Invaders-2
Primer juego con unity
Space Invaders 2 es un juego 2D desarrollado con Unity que rinde homenaje al clásico arcade de los años 80. Controlás una nave espacial y tu misión es defender la Tierra de oleadas de enemigos disparando misiles y esquivando sus ataques.

Este proyecto fue creado como práctica para aprender los fundamentos de desarrollo de videojuegos en Unity, incluyendo control de movimiento, disparos, colisiones y gestión de vidas.
